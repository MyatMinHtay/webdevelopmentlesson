HTML  ဆိုတာဘာလည်း

HTML ရဲ့ အရှည်ကောက်က Hyper Text Markup Language ဖြစ်ပြီးတော့ သူ့ကို Website တွေမှာ infromation တွေ ဖော်ပြပေးဖို့အတွက် အသုံးပြုပါတယ်။ HTML ကို Sir Tim Berners-Lee က 1991 မှာ ဖန်တီးခဲ့ပေမယ့် တရားဝင်ထုတ်ပြန်ခဲ့တာက 1995 ခုနှစ်မှာ HTML 2.0 အနေနဲ့ စထွက်ခဲ့တာဖြစ်ပြီး အခု ချိန်မှာတော့ HTML 5 ထိထွက်ရှိပြီးပါပြီ။

HTML က WEBSITE တွေတည်ဆောက်တဲ့နေရာမှာ အဓိက ကျတဲ့ အခန်းကဏ္ဍကနေပါဝင်နေပြီး browser ကို information တွေ ဘယ်လိုပြပေးရမလည်းဆိုတာကို သိရှိအောင် ပြုလုပ်ပေးပါတယ်

ဒါက html ကို စတင်ရေးသားမယ်ဆိုရင် လိုအပ်တဲ့ ကျောရိုးတစ်ခုပဲဖြစ်ပါတယ် html file တိုင်း ဒီ format ကို လိုက်နာရပါတယ်။ အဲ့ဒီမှာ <!DOCTYPE html> ဆိုတာသည် browser ကို ကိုယ်ရေးထားတဲ့ code တွေသည် html language ဖြစ်တယ်ဆိုတာကို ကြေညာပေးတာ declare လုပ်ပေးတာဖြစ်တယ် ပြီးရင် <html> (အဖွင့်) </html> (အပိတ်) ရှိမယ် အဲ့ အဖွင့်အပိတ်ထဲမှာမှ <head> (အဖွင့်) </head> အပိတ် <body> (အဖွင့်) </body> (အပိတ်) တွေပါရှိပါမယ်။ လူ့ခန္တာကိုယ်တစ်ခုလိုပေါ့ဗျာ ခေါင်းပိုင်းနဲ့ ခန္ဓာကိုယ်ပိုင်းပေါ့။
